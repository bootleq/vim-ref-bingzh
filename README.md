vim-ref-bingzh
==============

Bing dictionary (English/Chinese) source for [vim-ref][].

必应词典 智能手机版：
http://dict.bing.com.cn/?view=touch


Requirements
============

- [vim-ref][] by [thinca][]
- `curl` OR `wget`
- [webapi-vim][] by [mattn][]
- [wwwrenderer-vim][] by [mattn][]
- [OpenCC][] (optional, for 簡繁轉換 only) by [BYVoid][]


Status
======

Unstable, experimental.
